# E-Commerce User Journey Funnel Analysis

## Project Overview

This project analyzes customer behavior across an e-commerce platform to understand how users progress through the purchase journey and identify key drop-off points.

The analysis focuses on user interactions such as product views, wishlist additions, cart activity, and purchases. The goal is to uncover actionable insights that can improve conversion rates and reduce cart abandonment.

---

## Business Problem

E-commerce companies often struggle to understand why users do not complete purchases.

This project aims to answer questions such as:

- Where do users drop off in the funnel?
- Which product categories convert best?
- Which categories have the highest cart abandonment?
- Does product price impact conversion?
- How do different user segments behave?

---

## Key Insights

- Clothing generated the highest number of views but had the lowest conversion rate.
- Groceries achieved the highest conversion rate among all categories.
- Cart abandonment ranged from 64% to 69% across categories.
- Purchase behavior was evenly distributed across gender segments.
- High traffic did not always translate into high conversion.
