# SQL Queries

This folder contains SQL queries used to analyze user behavior across the e-commerce funnel.

## Analysis Covered

- Event Distribution Analysis
- Unique User Analysis
- Top Viewed Products
- Category Performance Analysis
- Purchase Analysis
- Conversion Rate Analysis
- Cart Abandonment Analysis
- Price & Rating Analysis
- User Segmentation Analysis
- City-wise Purchase Analysis
- Gender-wise Purchase Analysis

## Key KPIs

- Funnel Conversion Rate
- Category Conversion Rate
- Cart Abandonment Rate
- Product Views
- Purchases by Category
- Purchases by User Segment

## Files

- `funnel_analysis.sql` – Main SQL script containing all analysis queries used in the project.
